# 📖 React Todo List <a name="about-project"></a>

In this project, I have built a to-list app by using solely React and it's tools to practice all the tools and technologies that comes with React. You can add, edit, delete an item from the list and the items are stored in localStorage so that it stays until deleted.

## 💻 Getting Started <a name="getting-started"></a>

### Prerequisites

In order to run this project you need:
1. git
2. VS Code or any code editor
3. npm intalled


### Setup

Clone this repository to your desired folder:
```sh
  cd my-folder
  git clone git@github.com:rubydevi/react-todos.git
  cd react-todos
```

### Install

Install this project with:

```sh
 npm install
```

### Usage

To run this project in the browser, simply run the code below in your terminal:

```sh
 npm start
```

### Run tests

To run tests, run the following command:

```sh
 npx stylelint "**/*.{css,scss}"
```

```sh
 npx eslint "**/*.{js,jsx}"
```
